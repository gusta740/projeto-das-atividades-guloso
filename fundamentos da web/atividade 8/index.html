1. Exemplos de GET

  • Buscar a lista de avisos.
  • Abrir a página inicial.

2. Exemplos de POST

  • Cadastrar um novo aluno.
  • Adicionar um novo aviso.

3. O que significa 404

O código 404 significa que a página ou recurso não foi encontrado.

4. O que significa 201

O código 201 significa que um novo recurso foi criado com sucesso.

5. Exemplo de Request e Response

POST /avisos HTTP/1.1
Host: localhost:3000
Content-Type: application/json

{
  "titulo": "Reunião de Pais"
}

HTTP/1.1 201 Created
Content-Type: application/json

{
  "mensagem": "Aviso cadastrado com sucesso!"
}
